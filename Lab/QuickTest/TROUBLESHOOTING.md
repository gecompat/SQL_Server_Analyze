# Troubleshooting scope

- `RUNTIME_UNAVAILABLE`: verify the selected executable and its Compose plugin.
- `IMAGE_UNAVAILABLE`: verify registry reachability and the selected Microsoft tag.
- `PORT_CONFLICT`: choose another unprivileged host port before Install.
- `RESOURCE_LIMIT_EXCEEDED`: reduce the selected versions or resource profile.
- `SQL_NOT_READY`: inspect the local runtime logs; do not publish raw logs.
- `FRAMEWORK_INSTALL_FAILED`: inspect the ignored generated installer output for the affected synthetic instance.
