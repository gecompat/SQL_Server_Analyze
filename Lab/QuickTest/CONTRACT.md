# Contract summary

The quick-test lane is a separate, native Linux convenience environment. It
uses Docker or Podman through one PowerShell interface and may provision SQL
Server 2019, 2022, and 2025 independently. It does not replace the digest-bound
LAB-001 release lane or any Windows, Hyper-V, WSFC, FCI, failover, storage, or
network-fault evidence.
