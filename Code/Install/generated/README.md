# Generated installers

Die PowerShell-Builder unter `Code/Install` legen ihre eigenständig ausführbaren
SQL-Artefakte standardmäßig in diesem Verzeichnis ab. Dateien mit der Endung
`.generated.sql` werden nicht versioniert und dürfen nicht direkt bearbeitet
werden; maßgeblich bleiben die kanonischen SQL-Dateien und SQLCMD-Masterinstaller.
